Commandes relatives à la manipulation des images

Liste des commandes

picture_origin(id, origin)
Change l'origine d'une image
NomTypeDescription
idFixnumID de l'image
originFixnumOrigine de l'image, 0 = Haut gauche, 1 = centré, [x,y] = orienté autours de X,Y, par défaut, zéro
picture_show(id, name, *x, *y, *origin, *zoom_x, *zoom_y, *opacity, *blend_type)
Affiche une image à l'écran
NomTypeDescription
idFixnumID de l'image
nameStringNom de l'image (sans l'extension)
*xFixnumPosition en X de l'image (par défaut 0)
*yFixnumPosition en X de l'image (par défaut 0)
*originFixnumOrigine de l'image, 0 = Haut gauche, 1 = centré, [x,y] = orienté autours de X,Y, par défaut, zéro
*zoom_xFixnumZoom sur la largeur de l'image par défaut 100 (pour 100%)
*zoom_yFixnumZoom sur la hauteur de l'image par défaut 100 (pour 100%)
*opacityFixnumOpacité de l'image, par défaut 255 (de 0 à 255)
*blend_typeFixnumMode de fusion, par défaut 0, 0=Normal, 1=Soustraction, 2=Addition
picture_x(id, x)
Change l'axe X d'une image
NomTypeDescription
idFixnumID de l'image
xFixnumPosition en x de l'image, si aucun argument n'est passé, la commande renverra la position X de l'image
picture_y(id, x)
Change l'axe Y d'une image
NomTypeDescription
idFixnumID de l'image
xFixnumPosition en y de l'image, si aucun argument n'est passé, la commande renverra la position Y de l'image
