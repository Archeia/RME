    give_armor(id, amount, *include_equipement)
    Fait gagner à l'équipe l'armure référencé par son ID
    NomTypeDescription
    idFixnumId de l'armure
    amountFixnumNombre à donner. Si le nombre est négatif, l'objet sera retiré
    *include_equipementBooleanCe paramètre n'est utile que si on supprime des objets (ammount négatif). Si cette variable vaut true, les objets équipés seront déséquipés si l'inventaire n'en possède pas assez. Par défaut, il vaut false. Donc pas de déséquipement
